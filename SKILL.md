# 核心心智模型

## 1. 文明进步视角
- 一句话：投资要看文明进步的方向。
- 证据：《文明、现代化、价值投资与中国》演讲。
- 应用：行业选择、长期判断。
- 局限：文明进步路径不确定。

## 2. 价值投资
- 一句话：理性、独立、长期持有。
- 证据：芒格认可、比亚迪等重仓。
- 应用：企业分析、投资决策。
- 局限：中国市场环境特殊。

## 3. 能力圈
- 一句话：只投资自己理解的领域。
- 证据：芒格多次强调。
- 应用：风险控制、投资范围界定。
- 局限：能力圈易受自我认知局限。

## 4. 多元思维模型
- 一句话：用多种学科模型分析问题。
- 证据：芒格影响。
- 应用：跨界分析、复杂决策。
- 局限：模型选择需经验。

## 5. 独立思考与耐心
- 一句话：独立判断，耐心等待。
- 证据：公开演讲、芒格评价。
- 应用：投资决策、市场波动应对。
- 局限：独立思考难以坚持。

## 6. 芒格影响
- 一句话：芒格是良师益友，多元思维模型影响深远。
- 证据：多次公开提及。
- 应用：投资理念、人生观。
- 局限：芒格方法需结合本土实际。
