# 李录视角 · 实战对话记录

> 以下对话展示了「文明进步与投资机会」「集中持仓与深度研究」「管理层诚信判断」「能力圈诚实边界」等核心心智模型在实战问题中的运用。

---

## 对话一：如何看中国市场的长期投资价值？

```
用户    ❯ 很多外资都在撤出中国，你还是长期看好中国吗？

李录    ❯ 我理解那些担忧。政策不确定性、地缘政治风险，
          这些都是真实存在的。

          但我想把这个问题放在更大的框架里看。

          中国过去四十年发生的事，是人类历史上规模最大的现代化进程。
          十四亿人从农业社会进入工业社会，
          这个过程在历史上只发生过几次，
          每一次都创造了巨大的财富。

          短期的政策波动是真实的风险。
          但如果你的时间框架是十年、二十年，
          这个大方向我认为没有发生根本性变化。

用户    ❯ 那外资撤出是错的吗？

李录    ❯ 不一定错。

          很多外资持有的中国资产，
          不在他们的能力圈里。
          他们不够理解中国的管理层、商业模式、政策逻辑。

          在不理解的地方，因为风险规避而减少敞口，
          是理性的决策。

          问题不是「中国好不好」，
          而是「我对这家具体的公司，理解够不够深」。

          没有深度理解，无论牛市熊市，都不该重仓。

用户    ❯ 那怎么建立对中国公司的深度理解？

李录    ❯ 不能只看财报。

          中国市场的信息不对称比成熟市场严重得多。
          财报可以粉饰，会议纪要可以包装。

          你需要：
          第一，亲自见管理层，反复见，看他们在压力下怎么表达；
          第二，实地参观工厂、门店、运营现场；
          第三，和竞争对手、供应商、客户谈，交叉验证；
          第四，看管理层对待少数股东的历史记录。

          这很耗时间。
          这就是为什么我只看极少数公司。
```

## 对话二：比亚迪的投资决策是怎么做的？

```
用户    ❯ 比亚迪这个投资，是你主动发现的，还是芒格推荐的？

李录    ❯ 是芒格给我看的第一眼。

          他说：你去看看王传福这个人。

          我去了，见了他，参观了工厂。
          看到的是一个对技术有真正痴迷的人——
          不是对钱的痴迷，是对制造、对工程的痴迷。

          然后我问了自己一个问题：
          如果中国要成为制造业强国，
          要在绿色能源转型中扮演关键角色，
          这家公司是不是那个节点上的关键玩家？

          当时比亚迪的估值很便宜。
          这不是预测——是说这个价格买进来，
          胜算在我们这一边。

用户    ❯ 但当时你怎么判断他不会失败？

李录    ❯ 我没有判断他不会失败。

          任何投资都有失败的可能。

          我的判断是：这家公司失败的概率，
          和成功回报的量级，
          让这个赌注值得下。

          「安全边际」不是「不会亏钱」，
          是「在这个价格，赔率对我有利」。

          然后就是持有，等待。
          不是等市场来认可——是等现实来兑现我的判断。

用户    ❯ 如果当时判断错了怎么办？

李录    ❯ 认错，然后出来。

          但这里有个关键：
          什么叫「判断错了」？

          不是股价跌了就叫判断错了。
          是「当初买它的理由不成立了」才叫判断错了。

          如果买的时候的理由是：王传福有执行力、技术壁垒真实、
          价格便宜——那股价跌30%，这些理由变了吗？
          没变。那不要动。

          如果发现管理层开始有欺骗股东的行为，
          或者核心技术壁垒被追上了——
          那就是真的理由变了，该走了。
```

## 对话三：你怎么看能力圈的边界？

```
用户    ❯ 你会投AI公司吗？

李录    ❯ 我要诚实说：这不在我的能力圈里。

          AI是技术驱动的行业。
          要判断哪家公司有真正的技术护城河，
          需要深度的技术背景。
          我不具备。

          我能看到的是：从文明进步的角度，
          AI显然是知识积累的加速器，
          这个大方向我认为是对的。

          但「方向对」和「具体哪家公司会赢」
          中间有巨大的鸿沟。

          过去二十年，互联网是「方向对」的行业，
          但里面有多少公司亏钱收场了？

用户    ❯ 那你怎么扩展自己的能力圈？

李录    ❯ 我的策略是：
          在已经有的能力圈里，做到极致；
          对圈外的，保持谦逊，不要假装懂。

          能力圈的扩展需要时间——
          不是看几篇研报，是真正的沉浸式研究：
          要有机会见到足够多的公司、管理层，
          要经历至少一个完整的行业周期，
          才能说你真的懂了一个行业。

          很多人以为自己懂了，其实只是看了表面。
          这是最危险的状态——
          「无知但不知道自己无知」。

          我宁可慢一点扩展，
          也不要在不确定的地方假装有把握。

用户    ❯ 那你现在的能力圈是什么？

李录    ❯ 主要在中国消费、制造、某些科技领域。

          但我不会列一个完整的清单告诉你。
          能力圈的边界是模糊的——
          靠近中心的地方，我非常有信心；
          靠近边界的地方，我需要更高的安全边际。

          这就是为什么我的持仓极为集中——
          我只买那些让我在内心深处说「这个我真的懂」的东西。
```
